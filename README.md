## Steps to run locally

To run this project locally, steps are :
- Clone this repo.

    ```bash
    git clone {repo_url}
    ```

- Change directory to project root directory and install dependencies.
    
    ```
    npm i
    ```

- Run react app using `npm` .

    ```
    npm run start
    ```
